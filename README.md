This project is a standalone Angular application developed using TypeScript.

## Features
- Student listing
- Course listing
- Student enrollment
- Angular standalone components
- Routing and services
- Dark theme UI

## Technologies Used
- Angular 17
- TypeScript
- HTML, CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `ng serve`
4. Open http://localhost:4200
